# Raccoon NAI Bot

Telegram-бот для генерации изображений через NovelAI с простым UX: обычный пользователь, подписка 💎 Raccoon+ и отдельная 🛡 Moderation-панель.

## Роли и доступ

### 🌿 Ordinary

Обычный пользователь получает 10 обычных генераций в день. После дневного лимита бот расходует ✒ Ink. Базовые настройки доступны всем:

- 🎨 Model
- 📐 Orientation
- 📏 Resolution
- 🎲 Sampler
- 👣 Steps
- 🧲 CFG
- 🎲 Seed
- 🚫 Negative Prompt

Ограничения Ordinary:

- Steps: 1..28
- Images: 1
- Только безопасные размеры

### 💎 Raccoon+

Raccoon+ — платная подписка на 30 дней за 699 Telegram Stars. Она даёт:

- 100 обычных генераций каждый день
- 100 HQ генераций общим балансом на период подписки
- расширенные NovelAI-настройки
- Steps до 60
- несколько изображений
- большие размеры

Raccoon+ только разблокирует дорогие настройки; бот не включает их автоматически.

## HQ

HQ расходуется, если запрос использует хотя бы одно дорогое условие:

- Steps > 28
- большой размер
- несколько изображений

Списание: 1 изображение = 1 HQ. Например, 4 изображения стоят 4 HQ. Если генерация падает, HQ возвращается.

## Ink

Ink — запас платных обычных генераций после исчерпания дневного лимита. Одна обычная генерация стоит `INK_PER_GENERATION` Ink. Покупки Ink проходят через Telegram Stars.

## Магазин

Меню 🛒 Магазин содержит:

- ✒ Купить Чернила
- 💎 Купить Raccoon+
- ❤️ Поддержать проект

Успешная покупка Raccoon+ записывает срок `pro_access_until`, выдаёт `hq_balance = 100` и сохраняет платёжные метаданные Telegram.

## Moderation

Moderation не является режимом генерации. Доступ к модерации определяется только `ADMIN_IDS`.

Модераторы видят кнопку 🛡 Moderation в главном меню и могут открывать ту же панель командами совместимости:

- `/moderation`
- `/admin`
- `/xxx`

Модератор может переключать личную генерацию между 🌿 Ordinary и 💎 Raccoon+; это влияет только на его собственные настройки генерации и не меняет права модерации.

## Совместимость

Сохранены активные интеграции и инструменты:

- NovelAI
- Proxy
- Telegram Stars
- Ink
- Support
- Moderation channel
- Broadcasts
- History / Favorites
- Metadata import and retry
- Character+
- Site Clone / NovelAI debug
- Moderation tools

Устаревшие JSON-поля старых режимов безопасно читаются, но пользовательский UX больше не строится вокруг ArtRaccoon/Advanced/old PRO/provider/fal.ai modes.
